  1. viewport meta tag
    2. এইচটিএমএল (HTML) ডকুমেন্টের মেটা ট্যাগ যা ওয়েবসাইট মোবাইল এবং বিভিন্ন ডিভাইসে ভালোভাবে দেখানোর জন্য ব্যবহৃত হয়। ব্যাখ্যা: name="viewport" → এটা বলে দিচ্ছে যে, এটি ভিউপোর্ট (স্ক্রিনের দৃশ্যমান অংশ) সম্পর্কিত নির্দেশনা দিচ্ছে। content="width=device-width" → ব্রাউজারকে বলে, ওয়েবসাইটের প্রস্থ (width) যেন ডিভাইসের স্ক্রিনের প্রস্থের সমান হয়। অর্থাৎ, মোবাইল বা কম্পিউটারে দেখার সময় যেন স্বাভাবিক দেখায়। initial-scale=1.0 → এটা জুম লেভেল ঠিক করে, অর্থাৎ ওয়েবসাইট লোড হওয়ার সময় সেটি স্বাভাবিক স্কেল বা 100% জুম থাকবে।
